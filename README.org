* OSX Gotchas

  Use homebrew to get stuff that's not available in `nixpkgs` for Darwin:

  - virtualbox
  - chromium-freesmug

** 1password

   1password doesn't like Chromium.  Install the OSX app and go to preferences →
   advanced → uncheck verify browser code signature.
